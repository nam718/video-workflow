# 视频创作工作流

一个基于 Electron 的桌面应用，用来把剧本文案逐步转换成分镜、关键帧和视频素材。

## 功能概览

- 导入剧本并设置统一视觉风格
- 提取角色、场景、道具
- 生成参考图并关联到分镜
- 批量推理视频提示词
- 调用即梦、可灵等服务生成视频
- 时间轴预览与多版本对比

## 技术栈

- Electron
- Express
- Puppeteer / Playwright
- 原生 HTML + JS

## 本仓库包含

- 桌面应用源码
- Electron 本地服务端
- 即梦 / 可灵集成逻辑
- 构建脚本与默认提示词脚本

## 本仓库不包含

为了安全和体积控制，以下内容不会进入开源仓库：

- 打包产物，如 `dist/`
- 本地登录态、Cookie、项目历史
- 本地测试目录和临时文件
- 内置浏览器二进制文件
- 与本应用无关的旁支工具目录

## 开发启动

```bash
npm install
npm run dev
```

## 打包

```bash
npm run build:win
```

```bash
npm run build:mac
```

## 浏览器说明

项目默认优先使用：

1. 打包时附带的 Chromium
2. 系统已安装的 Chrome / Edge

开源仓库默认不提交 `chrome-win/` 二进制目录。如果你需要把浏览器一起打进 Windows 安装包，请在本地自行准备对应目录后再调整构建配置。

## 项目结构

```text
electron/   Electron 主进程与本地 HTTP 服务
src/        前端页面
shared/     共享脚本与提示词
scripts/    工作流脚本
jimeng/     即梦服务集成
kling/      可灵服务集成
```
