# 视频创作工作流 — 使用说明

## 一句话介绍

**从剧本到成片，6步搞定。** AI 自动拆解剧本、生成分镜、出图出视频，全程可控可编辑。

---

## 核心流程

```
📝 导入剧本 → 🔍 提取元素 → 🎨 生成参考图 → 🎬 剧本转分镜 → ✨ 推理+出图+出视频 → 🎞 时间轴预览
```

每一步的结果自动流入下一步，随时可手动编辑后重新生成。

---

## 步骤详解

### 📝 第1步 — 导入剧本

将小说、话本或故事文案导入工具。

- 支持 `.txt`、`.md` 文件导入，也可直接粘贴文本
- 设置整体**视觉风格**（写实、3D动漫、赛博朋克等），贯穿后续所有环节

### 🔍 第2步 — 提取角色·场景·道具

AI 自动分析剧本，提取三类核心元素：

- **角色**：外貌特征、服装风格、性格定位
- **场景**：空间布局、光线氛围、标志物件
- **道具**：关键特写物件的外观与剧情作用

提取结果可手动调整，确保后续生成的准确性。

### 🎨 第3步 — 生成参考图

为每个角色、场景、道具自动生成参考图片。

- 支持 **即梦** 和 **Gemini** 两种图片生成模型
- 可分类生成（仅角色 / 仅场景 / 仅道具）
- 不满意可重新生成或手动上传替代图
- 参考图将在后续步骤中自动关联到对应的分镜画面

### 🎬 第4步 — 剧本转分镜

AI 将完整剧本拆解为一段一段的分镜：

- 每段标注**背景、看点、场景、画面描述、对白、旁白**
- 自动识别分段边界，控制每段20-30字
- 支持**自定义转换预设**（⚙ 管理预设），可创建不同风格的转换模板
- 转换结果可逐段编辑调整

### ✨ 第5步 — 推理 + 出图 + 出视频

这是核心工作区，完成三件事：

#### 1. 推理提示词
点击 **▶ 推理全部提示词**，AI 为每个分镜生成：
- 画面提示词（用于 AI 绘图/视频生成）
- 对白/旁白文本
- 角色情绪标签

支持自定义推理预设（⚙ 管理预设），根据不同项目切换不同的提示词模板。

#### 2. 生成关键帧图片
点击 **🖼 生成全部分镜图**，为每个分镜生成一张关键帧参考图。

#### 3. 生成视频
- **🎥 并行生成视频** — 一次性提交所有分镜的视频生成任务
- **🚀 多开并发生成** — 启动多个即梦账号浏览器，自动负载均衡

视频参数可配置：

| 参数 | 选项 |
|------|------|
| 模型 | seedance 2.0 / 2.0-fast |
| 画面比例 | 16:9 / 9:16（竖屏） |
| 时长 | 4秒 ~ 15秒 |
| 生成数量 | ×1 / ×2 / ×3（每个分镜生成多个版本供挑选） |

#### 参考图智能关联
系统会自动匹配每个分镜中涉及的角色/场景/道具参考图，生成视频时一并参考，保证画面一致性。支持两种引用模式：
- **文中替换** — 在提示词中直接替换为参考图
- **开头引用** — 在提示词开头统一引用参考图

### 🎞 第6步 — 时间轴预览

所有分镜视频按顺序排列在时间轴上：

- 多轨道显示（当一个分镜有多个视频版本时）
- 连续播放预览，感受整体节奏
- 可切换轨道可见性，对比不同版本

## 功能亮点

### 🚀 多账号并发
注册多个即梦账号，多浏览器同时生成视频。单账号排满9个任务后自动切换下一个，全自动负载均衡。

### ⚙ 提示词预设系统
步骤4和步骤5均支持自定义 AI 提示词模板：
- 创建不同风格的预设（如写实风、动漫风、广告风）
- 支持变量占位符（`{{输入文案}}`、`{{characters}}` 等）
- 预设可随时切换，一键应用

### 📖 连续查看模式
全屏模式下连续预览相邻分镜的提示词和参考图，方便调整镜头衔接和叙事节奏。

### 📦 资产库侧边栏
侧边栏集中展示项目的所有参考图资产，方便快速查找和引用。

### 💾 项目管理
- 新建 / 打开 / 保存项目
- 所有数据（剧本、分析结果、参考图、分镜、视频）本地持久化
- 随时恢复到任意步骤继续工作

### 📐 节点式画布
各步骤以卡片形式呈现在可缩放、可平移的画布上，流程一目了然。支持自动布局和手动拖拽。

## API 配置

工具支持灵活的 AI 模型配置：
- 兼容任意 **OpenAI 格式 API**（含国内各家中转）
- 支持多组**配置档案**（Profile），可保存/切换/删除不同的 API 密钥和模型设置
- 文本推理和图片生成分别配置

---

## 适用场景

- 📱 短视频批量生产（抖音、快手、B站等）
- 🎬 网文/小说可视化（将文字故事变成视频）
- 📢 广告宣传片快速出片
- 🎓 教学演示视频制作
- 🎨 个人创作者的视频化表达

---

## 系统要求

- Windows 10/11（64位）
- 需要稳定的网络连接（用于 AI 推理和视频生成）
- 至少一个即梦平台账号（用于图片/视频生成）
